SRE/DevOps Learning App

Project Overview: This application demonstrates production-grade SRE and DevOps practices, including containerization, Kubernetes orchestration, automated CI/CD, and full-stack monitoring. The stack features a Flask backend, static frontend, Redis caching, and is deployable both locally and to AWS EKS. Monitoring is provided by Prometheus and Grafana, and all infrastructure is managed as code.
  • Frontend (HTML/JS) served by Nginx
  • Backend (Flask API) with health checks and metrics
  • Redis for caching and session storage
  • Docker Compose and Kubernetes deployment
  • Prometheus + Grafana monitoring stack
  • Automated CI/CD with GitHub Actions
